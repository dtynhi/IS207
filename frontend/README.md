# Uni Market Frontend (New)

React + Vite + Tailwind + Ant Design + TanStack Query scaffold for migration from legacy Pug frontend.

## Run

```bash
npm install
npm run dev --workspace frontend
```

## Current status

- Products listing/search/filter/detail screens are implemented
- Health page is wired to backend `/api/v1/health`
- Feature migration order is tracked in `docs/03-migration/feature-parity-checklist.md`
