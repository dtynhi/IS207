# Local Setup for Beginners

## Prerequisites

- Node.js 18+
- npm 9+
- PostgreSQL 14+
- MongoDB (only for legacy reference and ETL)

## Install

```bash
npm install
```

## Start new backend

```bash
npm run dev:backend
```

Backend URL: `http://localhost:4000`

## Start new frontend

```bash
npm run dev:frontend
```

Frontend URL: `http://localhost:5173`

## Start legacy app (reference)

```bash
npm run legacy:start
```

Legacy URL: `http://localhost:3000`
